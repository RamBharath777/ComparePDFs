# ComparePDFs

##  Prerequesites 

  > Min ***JAVA 8*** in PC <br>
  > Two PDFs to compare

## How It's works

    1. Dowload .exe file
    2. Double Click on .exe file
    3. Select Two PDfs and Click on Compare
    4. Result file will open with changes between two PDFs
